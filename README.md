# Tutorial de CRUD con PHP y HTML

## Descripción

Este repositorio contiene un ejemplo sencillo de una aplicación CRUD (Crear, Leer, Actualizar, Eliminar) utilizando PHP y HTML. El objetivo de este proyecto es proporcionar una guía básica para entender cómo interactúan estas tecnologías para gestionar datos.

## Contenido

*   **`index.html`**: Contiene el formulario para la creación de nuevos registros.
*   **`create.php`**: Procesa los datos enviados desde el formulario de `index.html`.

## Uso

1.  Abra el archivo `index.html` en su navegador.
2.  Rellene los campos del formulario.
3.  Haga clic en el botón "Crear".
4.  El archivo `create.php` procesará la información y mostrará el resultado.

## Próximos Pasos

Este es solo el primer paso (Crear). Las funcionalidades de Leer, Actualizar y Eliminar se implementarán en futuras actualizaciones.
